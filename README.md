# 柳比歇夫时间管理工具

基于柳比歇夫时间管理法的极简时间追踪工具。

## 特性

- 实时正向计时
- 强制属性补全（必须填写描述和分类才能记录）
- 本地数据存储（IndexedDB）
- 数据导出（JSON/CSV）
- 黑白极简风格
- 丝滑动画交互

## 分类

- 🔴 A类 - 核心产出（高浓度脑力劳动）
- 🟡 B类 - 基础事务（沟通、邮件、会议）
- 🟢 C类 - 生活休息（就餐、运动、放松）
- ⚫ D类 - 时间黑洞（无目的消耗）

## 技术栈

- React + Vite
- Zustand（状态管理）
- IndexedDB（本地存储）
- Recharts（图表）
- Framer Motion（动画）
- Tailwind CSS

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

构建后的静态文件可以直接部署到任意静态托管服务。
